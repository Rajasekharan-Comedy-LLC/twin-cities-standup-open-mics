  1. How do I say openmy.cc ? Just say "open mic", mic is spelled "mycc".
  2. How do I add a new mic? To add a new mic or make changes to this page, fork this repo and submit a pull request.

    This github help page tells you everything you need to know about creating forks and submitting a pull request.

    Alternately, you could send an email with the details of your new mic to info@openmy.cc to get added. Include the following details about the mic in your mail:

    • Name
    • Venue
    • Venue Address
    • Day and Time
    • Url
    • Booker(s)
    • Sign up details
    • any other notes you may want to include
